## 🚀 Quick Commands

```bash
# Compile, assemble, and run in one step
php src/phc.php exec examples/01_hello_world.php

# Build (compile + assemble)
php src/phc-new.php build test.php

# Individual steps
php src/phc.php compile test.php test.phas
php src/phc.php assemble test.phas test.phc
php src/phc.php run test.phc

# With verbose output
php src/phc.php exec test.php --verbose

# Run all tests
cd tests && php run_all_tests.php
```
